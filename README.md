# SpotExample

A POC example plugin for using Spot wheels generated in [spot-wheeler](https://github.com/Grkmr/spot-wheeler).

## Requirments

- [UV](https://github.com/Grkmr/spot-wheeler)

## DEV

To develop it run:

```sh
uv sync
```

Currently only usable on linux x86

## Installing

Run with uv:

```sh
uv run ocelecope build
```

or use the workflow for building in the ci
